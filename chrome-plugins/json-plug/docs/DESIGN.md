# JSON 格式化工具 - 页面布局说明

## 整体外观

这是一个深色主题的 JSON 格式化小工具，背景色为深灰接近黑色（#1e1e1e），整体给人 IDE/编辑器的感觉。

页面最上方有一个标题"JSON 格式化工具"，标题下方是主要的工作区域。

## 布局结构

整个工作区域由左右两个并列的面板组成，中间有 16 像素的间距。这两个面板是完全对称的，宽度各占一半。

### 左侧面板

左侧是输入区域，包含：
- 一个大文本框，用来粘贴或输入 JSON 字符串
- 两个并排的按钮："格式化"（蓝色）和"清空"（灰色）

### 右侧面板

右侧是输出区域，包含：
- 一个显示区域，用来展示格式化后的 JSON（带语法高亮）
- 底部一个绿色的"复制结果"按钮

## 重要设计约束

两侧的输入框/输出框必须保持完全相同的高度。即使输出区域为空，也应该填满可用空间。当内容超出时，输出区域可以上下滚动。

右侧的"复制结果"按钮必须和左侧的"格式化"、"清空"两个按钮处于同一水平高度。

## 按钮说明

- **格式化按钮**：点击后，将左侧输入的 JSON 字符串格式化，并在右侧显示结果
- **清空按钮**：清除左侧输入和右侧输出
- **复制结果按钮**：将右侧格式化后的 JSON 字符串复制到剪贴板，点击后文字会变成"已复制"

## 语法高亮颜色

格式化后的 JSON 使用代码编辑器风格的配色：
- 键名（key）：浅蓝色
- 字符串值：橙褐色
- 数字：浅绿色
- 布尔值和 null：蓝色
- 括号和冒号：灰色

## 错误提示

当 JSON 解析出错时，错误信息会显示在页面左下角的固定位置，以红色文字提示用户。
